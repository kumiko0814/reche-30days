【もりくみからの一言・写真の置き場所】

■ 日替わり写真
  ファイル名は日付で：  2026-06-20.jpg  のように置く（jpg/pngどちらでも）
  サイズは正方形がおすすめ（例 600×600px）。
  その日の写真が無ければ default.jpg → それも無ければ「森」アイコンを自動表示。

■ 既定写真（毎日同じでよければこれだけでOK）
  default.jpg  を置くと、日付指定が無い日はこれが出ます。

■ 一言テキストの更新
  morikumi.js の daily に1行足すだけ：
   "2026-06-20": { text: "今日のひとこと…", photo: "assets/morikumi/2026-06-20.jpg" },
